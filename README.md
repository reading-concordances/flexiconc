# FlexiConc

This repository combines the FlexiConc code and documentation with a clear separation:

- `/code`: Python package source and packaging metadata
- `/docs`: MkDocs source for documentation

## Current Version

FlexiConc version: **0.1.21** (from `/code/setup.py`).

## Installation

Install from PyPI:

```
pip install FlexiConc
```

PyPI package page: [https://pypi.org/project/FlexiConc/](https://pypi.org/project/FlexiConc/)

## Documentation

- Source (Markdown): `/docs/docs/index.md`
- Published site (GitHub Pages): [https://reading-concordances.github.io/flexiconc/](https://reading-concordances.github.io/flexiconc/)

To preview locally:

```
pip install mkdocs
mkdocs serve -f docs/mkdocs.yml
```
